# Unnamed Card Game

A Slay the Spire–style deckbuilder with Pokémon-style creature collecting,
built in [Godot](https://godotengine.org/) (4.7). Runs on mobile (touch) and
PC (mouse), forced landscape, targeting all standard aspect ratios.

## Structure

- `game/` — the Godot project (open this folder in the Godot editor).
- `docs/` — design notes and gameplay brainstorming.
- `art/wip/` — raw art in progress (e.g. Flux generations), not yet imported.
- `art/source/` — layered/source art files.

## Status

Early setup — project scaffolding and card browser are in progress.
See `docs/gameplay-notes.md` for the current gameplay premise and requirements.
